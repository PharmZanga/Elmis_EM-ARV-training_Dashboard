# MoH eLMIS EM and ARV Training Dashboard

Interactive React dashboard for monitoring facility reporting KPIs and linking EM/ARV training coverage back to facility performance.

## Sections

- Facility KPI page with reporting rate, timeliness, reporting status, trend, and facility tables.
- Training linkage page with experts, superusers, users, profession mix, and facility KPI linkage.
- Slicers for period, program, province, and district.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
